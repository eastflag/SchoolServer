* 실시간상담 입력하기--------------------------------------------------------------------------
who: 0: 학생, 1: 선생님
category
1: 성상담
2:학업상담
3:진로상담
4:심리상담
5:성장상담
6:흡연상담
http://aurasystem.kr:9000/admin/api/addConsult
{
	"content": "고민이 있어요",
	"category": 1,
    "who": 0,
	"member_id": 31
}
결과
{
    "result": 0,
    "msg": "success"
}
